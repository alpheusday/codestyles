# Code Styles

A collection of agent skills for Alpheus code styles.

## Usage

Install the skills with the [`skills`](https://github.com/vercel-labs/skills) CLI:

```sh
# npm
npx skills add alpheusday/codestyles

# pnpm
pnpm dlx skills add alpheusday/codestyles
```

## Skills

| Skill                                                        | Description                       |
| ------------------------------------------------------------ | --------------------------------- |
| [`alpheus-rust`](./skills/alpheus-rust/SKILL.md)             | Write Rust in Alpheus style       |
| [`alpheus-typescript`](./skills/alpheus-typescript/SKILL.md) | Write TypeScript in Alpheus style |
